## Quick Start

Use Pandas Tools to manipulate and analyze data with AI agents.

<Steps>
  <Step title="Install Dependencies">
    First, install the required packages:
    ```bash
    pip install praisonaiagents pandas
    ```
  </Step>

  <Step title="Import Components">
    Import the necessary components:
    ```python
    from praisonaiagents import Agent, Task, PraisonAIAgents
    from praisonaiagents.tools import pandas_tools
    ```
  </Step>

  <Step title="Create Agent">
    Create a data analysis agent:
    ```python
    data_agent = Agent(
        name="DataAnalyst",
        role="Data Analysis Specialist",
        goal="Analyze and transform data efficiently.",
        backstory="Expert in data manipulation and statistical analysis.",
        tools=[pandas_tools],
        self_reflect=False
    )
    ```
  </Step>

  <Step title="Define Task">
    Define the analysis task:
    ```python
    analysis_task = Task(
        description="Analyze sales data to identify trends and patterns.",
        expected_output="Statistical summary and key insights.",
        agent=data_agent,
        name="sales_analysis"
    )
    ```
  </Step>

  <Step title="Run Agent">
    Initialize and run the agent:
    ```python
    agents = PraisonAIAgents(
        agents=[data_agent],
        tasks=[analysis_task],
        process="sequential"
    )
    agents.start()
    ```
  </Step>
</Steps>

## Examples

### Basic Data Analysis Agent

```python
from praisonaiagents import Agent, Task, PraisonAIAgents
from praisonaiagents.tools import pandas_tools

# Create data analysis agent
data_agent = Agent(
    name="DataAnalyst",
    role="Data Analysis Specialist",
    goal="Perform comprehensive data analysis.",
    backstory="Expert in statistical analysis and data visualization.",
    tools=[pandas_tools],
    self_reflect=False
)

# Define analysis task
analysis_task = Task(
    description="Analyze customer demographics and purchase patterns.",
    expected_output="Customer segmentation analysis with insights.",
    agent=data_agent,
    name="customer_analysis"
)

# Run agent
agents = PraisonAIAgents(
    agents=[data_agent],
    tasks=[analysis_task],
    process="sequential"
)
agents.start()
```

### Advanced Analysis with Multiple Agents

```python
# Create data preparation agent
prep_agent = Agent(
    name="DataPrep",
    role="Data Preparation Specialist",
    goal="Clean and prepare data for analysis.",
    tools=[pandas_tools],
    self_reflect=False
)

# Create analysis agent
analysis_agent = Agent(
    name="Analyzer",
    role="Statistical Analyst",
    goal="Perform advanced statistical analysis.",
    backstory="Expert in statistical modeling and interpretation.",
    self_reflect=False
)

# Define tasks
prep_task = Task(
    description="Clean and prepare sales data for analysis.",
    agent=prep_agent,
    name="data_prep"
)

analysis_task = Task(
    description="Perform statistical analysis and generate insights.",
    agent=analysis_agent,
    name="statistical_analysis"
)

# Run agents
agents = PraisonAIAgents(
    agents=[prep_agent, analysis_agent],
    tasks=[prep_task, analysis_task],
    process="sequential"
)
agents.start()
```

## Common Patterns

### Data Analysis Pipeline
```python
# Data preparation agent
prep_agent = Agent(
    name="Preparer",
    role="Data Preparation",
    tools=[pandas_tools]
)

# Analysis agent
analyst = Agent(
    name="Analyst",
    role="Data Analyst"
)

# Define tasks
prep_task = Task(
    description="Clean and prepare data",
    agent=prep_agent
)

analysis_task = Task(
    description="Analyze prepared data",
    agent=analyst
)

# Run workflow
agents = PraisonAIAgents(
    agents=[prep_agent, analyst],
    tasks=[prep_task, analysis_task]
)
```
