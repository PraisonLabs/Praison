## Quick Start

Use Calculator Tools to perform mathematical computations with AI agents.

<Steps>
  <Step title="Install Dependencies">
    First, install the required package:
    ```bash
    pip install praisonaiagents
    ```
  </Step>

  <Step title="Import Components">
    Import the necessary components:
    ```python
    from praisonaiagents import Agent, Task, PraisonAIAgents
    from praisonaiagents.tools import calculator
    ```
  </Step>

  <Step title="Create Agent">
    Create a calculator agent:
    ```python
    calc_agent = Agent(
        name="Calculator",
        role="Mathematical Specialist",
        goal="Perform mathematical computations accurately.",
        backstory="Expert in mathematical calculations and analysis.",
        tools=[calculator],
        self_reflect=False
    )
    ```
  </Step>

  <Step title="Define Task">
    Define the calculation task:
    ```python
    calc_task = Task(
        description="Calculate financial metrics for quarterly report.",
        expected_output="Computed financial metrics with analysis.",
        agent=calc_agent,
        name="financial_calculations"
    )
    ```
  </Step>

  <Step title="Run Agent">
    Initialize and run the agent:
    ```python
    agents = PraisonAIAgents(
        agents=[calc_agent],
        tasks=[calc_task],
        process="sequential"
    )
    agents.start()
    ```
  </Step>
</Steps>

## Examples

### Basic Calculator Agent

```python
from praisonaiagents import Agent, Task, PraisonAIAgents
from praisonaiagents.tools import calculator

# Create calculator agent
calc_agent = Agent(
    name="MathExpert",
    role="Mathematical Specialist",
    goal="Perform calculations accurately and efficiently.",
    backstory="Expert in mathematical computations and analysis.",
    tools=[calculator],
    self_reflect=False
)

# Define calculation task
calc_task = Task(
    description="Calculate investment returns and compound interest.",
    expected_output="Detailed investment calculations with analysis.",
    agent=calc_agent,
    name="investment_analysis"
)

# Run agent
agents = PraisonAIAgents(
    agents=[calc_agent],
    tasks=[calc_task],
    process="sequential"
)
agents.start()
```

### Advanced Calculations with Multiple Agents

```python
# Create calculation agent
calculator_agent = Agent(
    name="Calculator",
    role="Math Calculator",
    goal="Perform complex calculations systematically.",
    tools=[calculator],
    self_reflect=False
)

# Create analysis agent
analysis_agent = Agent(
    name="Analyzer",
    role="Data Analyst",
    goal="Analyze and interpret calculation results.",
    backstory="Expert in data analysis and interpretation.",
    self_reflect=False
)

# Define tasks
calculation_task = Task(
    description="Calculate statistical metrics for dataset.",
    agent=calculator_agent,
    name="statistical_analysis"
)

analysis_task = Task(
    description="Analyze and interpret the statistical results.",
    agent=analysis_agent,
    name="result_analysis"
)

# Run agents
agents = PraisonAIAgents(
    agents=[calculator_agent, analysis_agent],
    tasks=[calculation_task, analysis_task],
    process="sequential"
)
agents.start()
```
