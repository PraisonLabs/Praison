## Quick Start

Create AI agents that can perform complex mathematical calculations, unit conversions, and financial computations!

<Steps>
  <Step title="Install Package">
    First, install the required package:
    ```bash
    pip install praisonaiagents
    ```
  </Step>

  <Step title="Import Required Components">
    Import the necessary components:
    ```python
    from praisonaiagents import Agent, Task, PraisonAIAgents
    from praisonaiagents.tools import (
        evaluate, solve_equation, convert_units,
        calculate_statistics, calculate_financial
    )
    ```
  </Step>

  <Step title="Create Math Agent">
    Set up your mathematical computation agent:
    ```python
    math_agent = Agent(
        name="MathAssistant",
        description="Expert in mathematical calculations and analysis",
        tools=[
            evaluate, solve_equation, convert_units,
            calculate_statistics, calculate_financial
        ]
    )
    ```
  </Step>
</Steps>

## Examples

### Basic Calculations Example

```python
from praisonaiagents import Agent, Task, PraisonAIAgents
from praisonaiagents.tools import evaluate, solve_equation

# Create agent
math_agent = Agent(
    name="MathAssistant",
    description="Expert in mathematical calculations",
    tools=[evaluate, solve_equation]
)

# Define tasks
calculation_task = Task(
    description="Calculate the value of: (3 + 4) * 2 - sqrt(16)",
    expected_output="The result of the mathematical expression",
    agent=math_agent
)

equation_task = Task(
    description="Solve the equation: x^2 - 5x + 6 = 0",
    expected_output="The solutions to the quadratic equation",
    agent=math_agent
)

# Initialize and run agents
agents = PraisonAIAgents(
    agents=[math_agent],
    tasks=[calculation_task, equation_task]
)
result = agents.start()
```

## Best Practices

<AccordionGroup>
  <Accordion title="Expression Handling">
    Handle expressions safely:
    ```python
    # Good - Using variables dictionary
    evaluate(
        "x^2 + y",
        variables={"x": 3, "y": 4}
    )
    
    # Good - Setting precision
    evaluate(
        "pi + e",
        precision=5
    )
    ```
  </Accordion>

  <Accordion title="Task Description">
    Write clear mathematical requirements:
    ```python
    # Good
    Task(description="""
        Calculate the following:
        1. Area of a circle with radius 5
        2. Convert the result to square inches
        3. Round to 2 decimal places
    """)
    
    # Too vague
    Task(description="Do some math")
    ```
  </Accordion>
</AccordionGroup>

## Common Patterns

### Multi-Step Calculations
```python
# Calculate area
area_task = Task(
    description="Calculate the area of a circle with radius 5",
    agent=math_agent
)

# Convert units
conversion_task = Task(
    description="Convert the area from square meters to square feet",
    agent=math_agent
)

# Format result
formatting_task = Task(
    description="Round the result to 2 decimal places",
    agent=math_agent
)
```

### Statistical Analysis Pipeline
```python
# Gather data
data_task = Task(
    description="Generate a dataset of 100 random numbers",
    agent=math_agent
)

# Calculate statistics
stats_task = Task(
    description="Calculate mean, median, mode, and standard deviation",
    agent=math_agent
)

# Visualize results
viz_task = Task(
    description="Create a summary of the statistical analysis",
    agent=math_agent
)
```

## Troubleshooting

<CardGroup cols={2}>
  <Card title="Expression Errors" icon="triangle-exclamation">
    Check expression syntax and variable definitions:
    ```python
    try:
        result = evaluate("x + y", variables={"x": 1})
    except Exception as e:
        print(f"Error: Missing variable 'y'")
    ```
  </Card>
  <Card title="Unit Conversion" icon="ruler">
    Ensure compatible unit types and valid unit names
    ```python
    # Valid conversion
    convert_units(100, "celsius", "fahrenheit")
    
    # Invalid conversion
    # convert_units(100, "meters", "celsius")
    ```
  </Card>
</CardGroup>
